# Store screenshots

Both Edge Add-ons and Chrome Web Store require 1–5 screenshots at
1280×800 px showing the extension in use. These aren't generated by
a script — they need to be real screenshots of the actual UI on your
deployed StockPilot. Here's what to capture:

## Shot 1 — The popup on a supplier tab (the headline shot)

1. Install the extension on your dev build.
2. Open `https://www.amazon.com` (or whichever supplier is showiest
   in your account — the "Ready" view looks best with a couple of
   suppliers in the dropdown).
3. Sign in to Amazon normally.
4. Click the StockPilot toolbar icon.
5. Screenshot the ENTIRE browser window at 1280×800 — the popup
   should be visible over the Amazon page. Highlight the popup's
   "Push cookies" button with a subtle arrow if you like.

Filename: `01-popup-on-amazon.png`

## Shot 2 — The signin wizard's Extension tab in StockPilot

1. In StockPilot, open any supplier's signin page.
2. Click the "Use browser extension (easiest)" tab.
3. Screenshot the full page at 1280×800. The download button should
   be prominent; the 3 numbered steps below it visible.

Filename: `02-install-walkthrough.png`

## Shot 3 — Success / confirmation

1. In the extension popup, after pushing cookies, the success view
   appears with the green ✓ and "Session saved" message.
2. Screenshot that view, ideally with the popup visible over a
   signed-in supplier page.

Filename: `03-success.png`

## Optional shot 4 — Connect-browser tab

1. Open `<your-stockpilot-url>/extension/connect` while signed in.
2. Screenshot the "This browser is linked" success page.

Filename: `04-linked.png`

---

## Mechanics

- 1280×800 is the native resolution for Chrome Web Store; Edge accepts
  the same.
- PNG format, under 4 MB each.
- No annotations or sales-y overlays; Microsoft rejects screenshots
  that look like ads. Pure UI captures only.
- On macOS: Cmd+Shift+4, then Space, then click the browser window —
  produces a clean screenshot with the drop shadow.
- On Windows: the built-in Snipping Tool (Win+Shift+S) → rectangular
  snip of the browser viewport.

Save the PNGs to this folder. The submission form uploads them one
at a time.
